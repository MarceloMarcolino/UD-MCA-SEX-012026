# Personal Webpage Project

## Overview
This project contains a personal webpage created using HTML and CSS. It showcases personal information, skills, interests, projects, experiences, and contact details.

## Project Structure
```
Aluno 1/
└── aula3_27022026/
    ├── nomeAluno.html      # HTML file for the personal webpage
    ├── CSS/
    │   └── style.css       # CSS file for styling the webpage
    └── README.md           # Documentation for the project
```

## Instructions to View the Webpage
1. Open the `nomeAluno.html` file in a web browser.
2. Ensure that the `style.css` file is located in the `CSS` folder for proper styling.

## Customization
- Replace placeholder text in `nomeAluno.html` with your personal information, including your name, RA, skills, interests, projects, experiences, and contact information.
- Add a personal image in the HTML file to enhance the visual appeal.
